Este projeto integrado tem como objetivo apresentar soluções técnicas para cinco desafios principais enfrentados pela TechMarket, aplicando conhecimentos multidisciplinares adquiridos ao longo do curso de Análise e Desenvolvimento de Sistemas. As soluções propostas abrangem: computação em nuvem para escalonamento de recursos, desenvolvimento de APIs RESTful para transações financeiras, programação web responsiva para dispositivos móveis, otimização de banco de dados com procedures e validação de formulários com JavaScript.
